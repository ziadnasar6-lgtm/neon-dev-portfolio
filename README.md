# 🚀 Futuristic AI & Software Developer Portfolio

A sleek, modern, and futuristic portfolio website built by **Ziad Nassar**. Designed specifically for software developers and AI engineers, this project features a stunning glassmorphic UI, cyber-inspired neon gradients, and smooth interactive animations.

---

## 🌟 Features

*   **Futuristic UI/UX:** Styled with the modern *Outfit* typeface, custom glow effects, and dynamic gradients perfectly suited for tech professionals.
*   **Fully Responsive:** Completely optimized for all device sizes, including mobile phones, tablets, and desktops.
*   **Scroll Animations:** Powered by the JavaScript `Intersection Observer API` to fade elements in seamlessly as the user scrolls.
*   **Dynamic Navbar:** Features a mobile hamburger menu and a smart scroll effect that shifts glassmorphism intensity as you navigate down.
*   **Interactive Contact Form:** Built-in form validation styles with dynamic UI state changes and button feedback upon sending.

---

## 🛠️ Tech Stack

*   **HTML5:** Structured using clean, semantic web standards.
*   **CSS3:** Styled using CSS Variables (`--variables`), modern layouts (Flexbox/Grid), custom animations (`@keyframes`), and responsive Media Queries.
*   **JavaScript (ES6+):** Handles DOM manipulation, mobile menu toggles, navbar transition triggers, and scroll animations.
*   **Google Fonts:** Integrated with the `Outfit` font family for a tech-forward look.

## 📝 Customization

*   **Personalize Content:** Open `index.html` to update your bio in the About section, and showcase your own custom projects in the Projects grid.
*   **Change Theme Colors:** You can easily swap the neon accents by modifying the CSS variables located in the `:root` selector of `styles.css`:
```css
    --accent-cyan: #00f3ff;   /* Primary accent color */
    --accent-purple: #bd00ff; /* Secondary accent color */
    ```
